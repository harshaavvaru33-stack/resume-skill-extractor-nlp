# Resume Skill Extractor (NLP)
This project extracts key information such as Name, Location, Organization, and Technical Skills from resume text using NLP techniques.
## Features
- Named Entity Recognition using spaCy
- Tokenization and preprocessing using NLTK
- Custom technical skills matching
- Terminal-based output
## Tech Stack
Python, NLTK, spaCy
## Use Case
Automated resume screening and skill analysis system.
